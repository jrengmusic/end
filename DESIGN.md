## I. Terminus

A terminal is an endpoint. The Latin *terminus* — a boundary, a limit, the place where a journey ends.

In computing, the terminal is where human intent meets machine execution. Commands enter. Output returns. The boundary between thought and process.

The word itself is an ending. Every terminal session is ephemeral — it begins, it runs, it ends. The machine forgets. The next session starts clean.

**Terminal is end.**

---

## II. Teletype

Before screens, before pixels, before memory — there was wire.

The teletype machine sat at the end of a telegraph line. It received electrical pulses and translated them into characters. Dots and dashes. Short and long. The most primitive encoding of language into signal.

Every modern terminal emulator emulates this machine. The PTY — pseudo-teletype — is the kernel interface. The file descriptor. The wire.

`/dev/tty` is the device. TTY is the abbreviation. Teletype is the origin.

**Dots and lines on a wire.**

Three colours. Three letters. Three layers radiating from a center point.

The signal emanates outward — from the nexus, through the display, into the ephemeral.

Read from center to edge. Decode by colour. The icon is the name.

## IV. Ephemeral Nexus Display

Three words. Each one precise.

**Ephemeral** — sessions are temporary. Processes spawn, execute, terminate. No session persists. The terminal holds nothing. Everything passes through.

**Nexus** — the convergence point. Editor, file manager, git client, language server, AI agent — all TUI tools meet here. One surface, many tools. The hub.

**Display** — the rendering surface. GPU-accelerated or CPU-rendered, runtime switchable. Text, code, diagrams, documents — everything is displayed. The screen is the product.

Three words that describe what the software does. Three dots in the icon that encode those words. The acronym that names both.

## V. Oblivion

Dark backgrounds. Cyan accents on near-black. Minimal chrome. Typography that serves data. Colour used sparingly — two brightness levels, not twelve.

The TET command interface from *Oblivion* (2013). Designed by GMUNK. A visual language built for monitoring, scanning, communicating. Dense information, clean hierarchy.

That visual language was trapped in film. Beautiful but non-functional. You cannot use a movie UI.

Until the rendering surface exists. Until the font is yours. Until the palette maps to configuration keys. Until the terminal is the interface.

**The design language becomes functional when the tool is real.**

## VI. Display

The font ships with the terminal. Not downloaded, not configured, not fallback — embedded in the binary.

Display Mono for the terminal. Display for the document viewer. Geometric. Clean. The same family, proportional and monospaced.

A typeface designed before the terminal existed. Evolved alongside it. The font and the software grew together — not planned, not forced. The tool shaped the type. The type shaped the tool.
